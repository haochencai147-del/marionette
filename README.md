# Marionette
Unified startup method: Run `npm start` in the project directory, and then open `http://localhost:3000` in the browser. Camera permission is required.
Core files: `index.html`, `style.css`, `sketch.js`.
This project is an interactive web-based system that uses real-time hand tracking to map users’ gestures onto pixel characters, enabling a dual-character battle experience controlled through embodied interaction.
